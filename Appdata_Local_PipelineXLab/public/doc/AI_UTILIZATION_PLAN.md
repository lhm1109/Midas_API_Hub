# PipelineXLab AI 활용 기획서

## 1. 개요

PipelineXLab에 AI를 통합하여 사용자 생산성과 경험을 향상시키는 방안을 제안합니다.

---

## 2. AI 배치 전략 비교

| 위치 | 장점 | 단점 | 적합 용도 |
|------|------|------|-----------|
| **Flow Editor (추천)** | 작업 컨텍스트 활용, 노드/플로우 직접 조작 | Flow Editor에 의존성 | 자동화, 코드 생성, 분석 |
| **Launcher** | 독립적 동작, 시스템 관리 | 플로우 컨텍스트 제한적 | 설정 도우미, 문서 검색 |
| **하이브리드** | 양쪽 장점 활용 | 구현 복잡도 증가 | 전체 통합 경험 |

### ✅ 권장: **Flow Editor 중심 + Launcher 보조**

## 3. Flow Editor AI 기능 (핵심)

### 3.1 AI Assistant Panel
```
┌─────────────────────────────┐
│ 🤖 AI Assistant             │
├─────────────────────────────┤
│ "RC 보 설계 플로우 만들어줘" │
│                             │
│ [생성 중...]                │
│                             │
│ ✅ NumberInput 3개 생성     │
│ ✅ Python 노드 연결         │
│ ✅ Display 출력 연결        │
└─────────────────────────────┘
```

**기능:**
- 자연어로 플로우 자동 생성
- 선택된 노드 설명 요청
- Python 코드 자동 작성
- 오류 분석 및 수정 제안

### 3.2 Smart Node Suggestions
- 현재 플로우 분석 → 다음 노드 추천
- "이 분석에는 Display 노드가 필요합니다"
- 연결 가능한 노드 하이라이트

### 3.3 Code Generation
- Python 노드 내 AI 코드 자동완성
- 수식 → SymPy 변환
- OpenSeesPy 보일러플레이트 생성

### 3.4 Flow Optimization
- 비효율적인 연결 감지
- 중복 노드 병합 제안
- 실행 순서 최적화

## 4. Launcher AI 기능 (보조)

### 4.1 Quick Actions (AI Tab)
```
[💬] "OpenSeesPy로 지진해석하려면?"
     → 관련 예제 플로우 추천
     → 필요한 컴포넌트 목록
     → 문서 링크 제공
```

### 4.2 Troubleshooting
- 서비스 오류 자동 진단
- Docker 문제 해결 가이드
- 로그 분석 및 해결책 제안

### 4.3 Component Discovery
- "철근 배근 계산하는 컴포넌트 있어?"
- Marketplace 검색 보조
- 컴포넌트 사용법 설명

## 7. 예상 사용 시나리오

### 시나리오 1: 신규 사용자
```
사용자: "RC 기둥 설계 워크플로우 만들어줘"

AI: "RC 기둥 설계 플로우를 생성합니다:
     1. NumberInput (폭, 높이, fc, fy)
     2. Python 노드 (단면 계산)
     3. OpenSeesPy 모델
     4. Display 결과 출력
     
     [플로우 생성하기]"
```

### 시나리오 2: 코드 작성
```
Python 노드 내:

사용자: "# 단면2차모멘트 계산해줘"

AI 자동완성:
    def calculate_moment_of_inertia(b, h):
        """직사각형 단면 2차 모멘트"""
        return (b * h**3) / 12
```

### 시나리오 3: 오류 해결
```
오류: "Connection refused: PostgreSQL"

AI: "PostgreSQL 연결 오류입니다.
     1. Docker가 실행 중인지 확인
     2. Database 서비스 상태 확인
     3. 포트 5432가 사용 가능한지 확인
     
     [Docker 시작하기] [서비스 재시작]"
```
